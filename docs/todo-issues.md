# 待人工确认事项

以下事项不属于当前代码小范围修复内容，但课堂提交前建议确认。

1. 课程设计报告和答辩 PPT 不在本仓库内，需要按老师给的 Word 格式单独整理。
2. MetaMask 账户导入和授权需要人工操作，不能通过自动测试完全替代。
3. 每次重启 Ganache 并重新部署后，合约地址会变化；演示前需要重新执行 `npm run migrate`，并确认前端读取的是最新 `campusRentalAddress.json`。
4. 本项目前端连接的是本地 Ganache，不建议直接连接真实主网或提交任何真实私钥、助记词、`.env` 文件。
5. 如果老师要求“租赁超时自动扣款/按实际归还时间计费”，当前合约未实现该扩展；当前实现按选题一基础要求，在租赁时锁定租金和押金，确认归还时退押金并结算租金。
